MoodDiary – Дополнителна задача ПНУВ

Ова е веб‑апликација изработена како дополнителна задача по предметот ПНУВ.  
Апликацијата е инспирирана од HBSC податоците и има за цел да промовира психолошка добросостојба кај младите.

Функционалности
- Внесување дневно расположение со коментар
- Историја на внесени записи
- График за визуелизација на расположенијата
- Чување податоци во localStorage
- Пример записи (Среќен, Тажен, Енергичен) за демонстрација

Технологии
- HTML, CSS, JavaScript
- Chart.js за графици

Како да се користи
1. Отвори ја страницата во прелистувач.
2. Внеси расположение и коментар преку формата.
3. Записот ќе се додаде во историјата и графикот ќе се ажурира.
4. Податоците се зачувуваат локално (localStorage).

Цел
Да се поттикне свесност за менталното здравје и да се овозможи едноставен начин младите да го следат своето расположение.
